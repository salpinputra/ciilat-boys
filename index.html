Love Coffee

I created this website by following a tutorial from my online lecturer named Sandhika Galih, I followed all the UI/UX displays but I added some additions from the video tutorial that I followed. I also added a login feature using the PHP programming language to verify user data, that only registered users get more features from the website and, I also added an admin_dashboard website, to make it easier for admins to perform CRUD methods (Create, Read, Update, Delete).

Form Login

This is a login form that I created using HTML, CSS and I use PHP to verify the data, that only registered users can enter the website and if the user enters the wrong username and password, it will display an error message and will also display that the user must register if the user has not registered. I use it with the Javascript programming language, I also use hashing to maintain the security of user privacy

Book animation
